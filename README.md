# 🌐 Basic Network Sniffer

## 📌 Overview

Basic Network Sniffer is a Python-based cybersecurity project developed using the Scapy library. It captures live network packets and extracts useful information such as source and destination IP addresses, protocols, ports, timestamps, and packet lengths. The project also exports captured packet information to a CSV file and saves packets in PCAP format for further analysis.

---

## 🚀 Features

- Capture live network packets
- Display Source IP Address
- Display Destination IP Address
- Detect TCP, UDP, and ICMP protocols
- Display Source and Destination Ports
- Display Packet Length
- Display Timestamp
- Export captured packet details to a CSV file
- Save captured packets as a PCAP file

---

## ⚙ Installation

Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/Basic-Network-Sniffer.git
```

Move into the project folder:

```bash
cd Basic-Network-Sniffer
```

Install the required library:

```bash
pip install -r requirements.txt
```

Run the project:

```bash
python sniffer.py
```
